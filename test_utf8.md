## Acentos y caracteres especiales

Esta línea tiene acentos: á é í ó ú ñ ü

### Símbolos

- Guiones largos: esto -- aquello
- Comillas: "comillas dobles" y 'comillas simples'
- Elipsis: ...

## Mezcla con formato

Texto con **negritas y acentos**: árbol, corazón, murciélago.

Texto con *itálica y acentos*: lápiz, carácter, difíciles.

Código inline con acentos: `código_de_ejemplo`.

> Cita con acentos: más de cien mil años de historia.

```bash

cat /var/log/sistema.log | grep "acción"
```
